            BadWork
===============================

Esta extensión de Firefox servirá para restringir el acceso de los trabajadores. Teniendo roles de administrador y usuario.
El administrador podrá:
	1. Bloquear páginas.
	2. Desbloquear páginas.
	3. Listar las páginas bloqueadas.
	4. Crear y cambiar una contraseña de acceso.

			Carpetas
===============================

 * dist ------> contiene la extensión comprimida y firmada por Mozilla pudiendo instalarla en el navegador Firefox.
 * docs ------> contiene la documentacion en PDF y un archivo de presentación
 * src  ------> contiene el codigo fuente de la aplicación
